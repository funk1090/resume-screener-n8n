# 🤖 AI Resume Screener — n8n + Ollama

An automated resume screening workflow built with **n8n** and **Ollama (local LLM)** that evaluates candidates against a job description, ranks them by match percentage, and delivers results via email — all running locally with no external AI API costs.

---

## 📌 Overview

This workflow automates the resume screening process by:

1. Accepting a **Job Description** and multiple **Resumes** as PDF uploads via a web form
2. Extracting text from each document
3. Using a **local AI model (Llama3 via Ollama)** to evaluate each candidate against the job description
4. Generating a **match score and recommendation** for each candidate
5. Exporting results to a **CSV file**
6. Sending the results by **email with the CSV attached**

---

## 🧰 Tech Stack

| Tool | Purpose |
|------|---------|
| [n8n](https://n8n.io/) | Workflow automation (self-hosted) |
| [Ollama](https://ollama.com/) | Local LLM runner |
| [Llama3](https://ollama.com/library/llama3) | AI model for resume evaluation |
| Gmail | Email delivery of results |

> ✅ **Runs 100% locally** — no OpenAI or external AI API required. Only Gmail is used as an external service for delivery.

## 🔄 Workflow Architecture

```
[Web Form: Upload PDFs]
        │
        ├──→ [Extract Job Description text]
        │
        └──→ [Split & Extract Resume text (one per candidate)]
                        │
              [Combine JD + Each Resume]
                        │
              [AI Agent - Ollama/Llama3]
              Evaluates match per candidate
                        │
              [Clean & Parse JSON output]
                        │
              [Calculate Recommendation]
              Best Fit / Average Fit / Not Fit
                        │
              [Export to CSV]
                        │
              [Send Email with CSV attachment]
```

## ⚙️ Requirements

- [n8n](https://docs.n8n.io/hosting/) self-hosted (Docker recommended)
- [Ollama](https://ollama.com/) installed locally
- Llama3 model pulled: `ollama pull llama3`
- Gmail account with OAuth2 configured in n8n

---

## 🚀 How to Use

**1. Import the workflow**
- Open your n8n instance
- Go to **Workflows → Import**
- Upload `Resume_Screener.json`

**2. Configure credentials**
- Set up your **Ollama** connection (default: `http://localhost:11434`)
- Set up your **Gmail OAuth2** credentials in n8n
- Update the email address in the `Send Email` node

**3. Activate and run**
- Activate the workflow
- Open the form URL generated by n8n
- Upload your **Job Description PDF** and one or more **Resume PDFs**
- Wait for the email with results
